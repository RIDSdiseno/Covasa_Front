# COVASA Front

Frontend web para COVASA (gestión de cobranza, ventas, inventario y conciliación).

## Stack

- React + TypeScript + Vite
- Tailwind CSS
- React Router

## Requisitos

- Node.js 20+
- npm

## Instalar

```bash
npm install
```

## Ejecutar

```bash
npm run dev
```

## Scripts útiles

```bash
npm run lint
npm run build
npm run preview
```

## Login (demo)

Por ahora el login es de demostración (no hay backend): valida formato de correo y exige contraseña no vacía, pero no la verifica.
